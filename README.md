## Feature Flags

| Syntax       | Description |                                                                      Default  |
| :---         |    :-----------------------------------------------------------------------:   | --------:  |
| `query`      | Allows for the usage of extra functions to fetch singular queries.                   | YES  |
| `fetcher`    | Allows for the usage of a fetcher that will cache student data in a struct.          | YES  |
